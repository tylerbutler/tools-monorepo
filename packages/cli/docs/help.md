`tbu help`
==========

Display help for tbu.

* [`tbu help [COMMAND]`](#tbu-help-command)

## `tbu help [COMMAND]`

Display help for tbu.

```
USAGE
  $ tbu help [COMMAND...] [-n]

ARGUMENTS
  COMMAND...  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for tbu.
```
