`sail help`
===========

Display help for sail.

* [`sail help [COMMAND]`](#sail-help-command)

## `sail help [COMMAND]`

Display help for sail.

```
USAGE
  $ sail help [COMMAND...] [-n]

ARGUMENTS
  [COMMAND...]  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for sail.
```
